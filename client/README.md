# Product & Category Client

React frontend for the Product & Category API.

## Setup

```bash
npm install
npm start
```

Runs on http://localhost:3000

## Configuration

Create `.env` for custom API URL:

```env
REACT_APP_API_URL=http://localhost:5000/api
```

## Build

```bash
npm run build
```

Deploy the `build` folder to Vercel or Netlify.
